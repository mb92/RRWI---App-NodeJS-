URL Methods Params Descriptions Example
URL Methods Params Descriptions Example
/connect/P/B POST connect / <X>
P - name of port (string)
B ∈ Z (Integer) 		Connect to printer
connect <port> <baudrate>
If port and baudrate are not specified, connects to first detected port at 115200bps
 /connect/portname/14400 (turn off heating)
/bedtemp/X POST bedtemp / <X>
X [oC]
X ∈ [0, max] ∪ {'abs', 'pla', 'off'} X (String) 		Sets the bed temperature to the value entered. Enter either a temperature in celsius or one of the following keywords abs(110), off(0), pla(60) bedtemp 120 bedtemp 0 /bedtemp/off (turn off heating)
/bedtemp/abs (set temp for abs 110)
/bedtemp/120 (set 120 oC)
/bedtemp/0 (set the same as off value)
/settemp/X POST settemp / <X>
X [oC]
X ∈ [0, max] ∪ {'abs', 'pla', 'off'} X (String) 		Sets the hotend temperature to the value entered. Enter either a temperature in celsius or one of the following keywords abs(230), off(0), pla(185) /settemp/off (turn off heating)
/settemp/abs (set temp for abs 110)
/settemp/120 (set 120 oC)
/settemp/0 (set the same as off value)
/exit GET - Disconnects from the printer and exits the program. -
/home/A POST home / <A>
A ∈ {'x', 'y', 'z', 'xy', 'e', 'xyze', 'xyz', 'xz, 'yz'} A (String) 		Homes the printer
null (without params) - homes all axes and zeroes the extruder(Using G28 and G92)

x - homes x axis only (Using G28)
y - homes y axis only (Using G28)
xy - homes x and y axes (Using G28)
z - homes z axis only (Using G28)
e - set extruder position to zero (Using G92)
xyze - homes all axes and zeroes the extruder (Using G28 and G92)
 /home
/home/x
/home/xy
/move/A/B/C POST move / <A> / <B> / <C>
A ∈ {'x', 'y', 'z'} (String)
B ∈ [0, max] (Integer)
C ∈ [0, max] (Integer) 		Move an axis. Specify the name of the axis and the amount.
move X 10 will move the X axis forward by 10mm at 3000 mm/min (default XY speed)
move Y 10 5000 will move the Y axis forward by 10mm at 5000mm/min
move Z -1 will move the Z axis down by 1mm at 100 mm/min (default Z speed)
Common amounts are in the tabcomplete list.
 /move/X/10
/move/X/10/3000
/move/Y/20
/resume GET - Resumes a paused print. -
/upload GET - Uploads a gcode file to the sd card. -
/set/A/B POST set / <A> / <B>
A ∈ {'x', 'y', 'z'} (String)
B ∈ [0, max] (Integer)
C ∈ [0, max] (Integer) 		Set variable: set Show variable: set 'set' without arguments displays all variables /set/X/10
/set/X/10
/set/Y/20
/extrude/A/B POST extrude / <A> / <B>
A ∈ (-∞, ∞+) - length of fiament [mm] (Integer)
B ∈ [-max;, max] - acceleration [mm/min] (Integer)
 Extrudes a length of filament, 5mm by default, or the number of mm given as a parameter
extrude - extrudes 5mm of filament at 300mm/min (5mm/s)
extrude 20 - extrudes 20mm of filament at 300mm/min (5mm/s)
extrude -5 - REVERSES 5mm of filament at 300mm/min (5mm/s)
extrude 10 210 - extrudes 10mm of filament at 210mm/min (3.5mm/s)
of mm given as a parameterh of filament, 5mm by default, or the number
 /extrude/-5
/extrude/10
/extrude/15/200
/load GET - Loads a gcode file (with tab-completion) -
/off GET - Turns off everything on the printer -
/reverse/A/B POST reverse / <A> / <B>
A ∈ (-∞, ∞+) - length of fiament [mm] (Integer)
B ∈ [-max;, max] - acceleration [mm/min] (Integer)
 Reverses the extruder, 5mm by default, or the number of mm given as a parameter
reverse - reverses 5mm of filament at 300mm/min (5mm/s)
reverse 20 - reverses 20mm of filament at 300mm/min (5mm/s)
reverse 10 210 - extrudes 10mm of filament at 210mm/min (3.5mm/s)
reverse -5 - EXTRUDES 5mm of filament at 300mm/min (5mm/s)
 /reverse/-5
/reverse/10
/reverse/15/200
/gcodes GET - Gcodes are passed through to the printer as they are. -
/ls GET - Lists files on the SD card -
/pause GET - Pauses a running print -
/disconnect GET - Disconnects from the printer -
/gettemp GET - Read the extruder and bed temperature. -
/print GET - Send a loaded gcode file to the printer. Load a file with the load command first. -
/eta GET - Displays estimated remaining print time. -
/monitor GET - Monitor a machine's temperatures and an SD print's status.
monitor - Reports temperature and SD print status (if SD printing) every 5 seconds
monitor 2 - Reports temperature and SD print status (if SD printing) every 2 seconds 		-
/reset GET - Resets the printer. -
/sdprint/N POST N - file name (String) Print a file from the SD card. Tab completes with available file names. sdprint filename.g sdprint filename.gcode
